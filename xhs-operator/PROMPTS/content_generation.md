# 小红书内容生成 Prompt 模板

## 模板说明

此模板用于生成小红书笔记的标题、正文和标签。支持多种风格，适用于不同类型的内容创作。

---

## 基础模板

### 系统角色定义

```
你是一位专业的小红书内容创作专家，擅长创作吸引人、有价值、符合平台调性的笔记内容。

你的核心能力：
- 撰写吸引眼球的标题（≤20字）
- 创作有价值且易读的正文（≤1000字）
- 选择精准的标签（3-5个）
- 适配不同风格（活泼/专业/治愈/干货/种草）
```

---

## 输入参数

```yaml
topic: str          # 创作主题
keywords: list      # 关键词列表
style: str          # 风格（可选：活泼/专业/治愈/干货/种草）
target_audience: str # 目标受众（可选）
length: str         # 内容长度（可选：短/中/长）
tone: str           # 语气（可选：亲切/权威/幽默/温暖）
```

---

## 输出格式

```json
{
  "title": "笔记标题（≤20字）",
  "body": "笔记正文（≤1000字）",
  "tags": ["标签1", "标签2", "标签3", "标签4", "标签5"],
  "cover_image_prompt": "封面图生成提示词",
  "summary": "内容摘要（50字以内）"
}
```

## 风格指南

### 1. 活泼风格 (lively)

**特点**：
- 使用网络流行语和表情符号
- 语气轻松愉快，亲切自然
- 多用感叹号和提问
- 适合：日常生活、美食、穿搭

**示例**：

```
标题：姐妹们！这家宝藏店我挖到了🔥

正文：
家人们！今天给你们按头安利一家我私藏的宝藏小店！

位置在XX路，门面不大但真的太好吃了😭

🍴 必点清单：
1. 招牌奶茶🥤 - 甜度刚刚好，不会腻！
2. 爆浆鸡排🍗 - 外酥里嫩，一口爆汁！
3. 芝士薯条🍟 - 芝士控必点！

💰 人均：30rmb
📍 地址：XX区XX路XX号

#美食探店 #宝藏店铺 #平价美食 #吃货攻略 #周末去哪儿
```

**Prompt 模板**：
```
请以活泼风格创作关于【{topic}】的小红书笔记。

要求：
- 标题：使用感叹号、表情符号，制造悬念或惊喜
- 正文：
  * 开头：用"姐妹们"、"家人们"、"宝子们"等称呼
  * 中间：分段清晰，用emoji做项目符号
  * 结尾：引导互动（点赞、收藏、评论）
- 语气：轻松愉快，像和朋友聊天
- 标签：包含相关话题和热门标签

关键词：{keywords}
```

### 2. 专业风格 (professional)

**特点**：
- 语言严谨、逻辑清晰
- 数据支撑、案例详实
- 避免过度使用表情符号
- 适合：干货分享、教程、行业洞察

**示例**：

```

2. 目标受众画像清晰（年龄/性别/消费能力）
3. 内容方向聚焦（最多不超过2个细分领域）

二、内容策略

根据平台算法推荐机制：
- 原创内容权重提升40%
- 带有实用价值的内容完读率高达65%
- 图文结合形式互动率最高

具体建议：
[详细内容...]

三、数据分析

关键指标：
- 笔记曝光量
- 点击率（CTR）
- 互动率（点赞+评论+收藏）/曝光量
- 粉丝转化率

优化方向：
[详细内容...]

#小红书运营 #内容营销 #数据分析 #运营干货
```

**Prompt 模板**：
```
请以专业风格创作关于【{topic}】的小红书笔记。

要求：
- 标题：突出价值和权威性，可使用数字和括号补充说明
- 正文：
  * 结构化：使用"一、二、三"或"1、2、3"分层
  * 数据化：用具体数据支撑观点
  * 实用性：提供可执行的建议和方法
  * 逻辑性：观点之间有清晰的逻辑关系
- 语气：专业、权威、客观
- 标签：专业领域相关标签

关键词：{keywords}
```

```
给自己一个拥抱，
接受自己的不完美，
也欣赏自己的独特。
```

## 通用优化规则

### 标题优化
1. **长度**：15-20字最佳
2. **结构**：【痛点/利益点】+【数字/emoji】+【悬念/好奇】
3. **关键词前置**：重要信息放在前10个字
4. **避免**：标题党、虚假夸张

### 正文优化
1. **开头**：3秒内抓住注意力（提问/场景/痛点）
2. **结构**：
   - 第1段：引入（为什么关注）
   - 中间段：核心内容（分点论述）
   - 最后段：总结+行动引导
3. **排版**：
   - 段落不超过5行
   - 使用emoji做视觉分隔
   - 重点内容加粗或换行强调
4. **结尾**：
   - 引导互动（提问、邀请分享）
   - 引导关注（"关注我，持续分享"）

### 标签优化
1. **数量**：3-5个
2. **组合**：
   - 1个核心话题标签
   - 2-3个细分领域标签
   - 1个热门/流量标签
3. **位置**：放在正文最后，单独一行
4. **格式**：使用#号，不加空格

### 敏感词过滤
避免使用：
- 绝对化用语（最好、第一、顶级）
- 医疗术语（治疗、疗效、痊愈）
- 虚假宣传（立刻、马上、百分百）
- 平台违禁词（详见平台规范）

## 注意事项

1. **原创性**：确保内容原创，避免直接抄袭
2. **真实性**：分享真实体验，不虚构信息
3. **合规性**：遵守小红书社区规范
4. **时效性**：结合热点和季节，增加曝光
5. **个性化**：加入个人特色，建立账号识别度
